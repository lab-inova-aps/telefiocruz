Authorize the use of Zoom

To authorize Telefiocruz application to create meetings with your Zoom account, follow these steps:

  1. Access the application (https://telefiocruz.aplicativo.click) and sign in with your username and password
  2. Click on "Configurar Zoom / Configure Zoom" shortcut
  3. Read the message that will be displayed and submit the form
  4. Authenticate yourself on Zoom site
  5. Consent the authorization of creating meeting in your Zoom account
  6. Observe the "Abrir Sala / Create Meet" shortcut in main page

Meet Creation

To create a meeting within the application, follow the steps below:

  1. Click on "Abrir Sala / Create Room" shortcut
  2. Enable the camera/microphone and join the room
  3. Copy the page URL, send it to the patient (by e-mail, Whatsapp or SMS) and wait for him/her
  4. After the call, leave the room and go back to home page

Revoking the use of Zoom

To revoke the authorization mentioned above, follow these steps:

  1. Click the "Configurar Zoom / Configure Zoom" shortcut
  2. Read the message that will be displayed and submit the form
  3. Observe that the "Abrir Sala / Create Room" shortcut disappeared main page
